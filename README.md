==Atmosphere==
After the Atmosphere Setup when asked for inserting next CD, cancel and copy the Data files in their specified file directory. After that, 'register' the location with SpectraMove.

==Trilogy==
After the Trilogy Setup when asked for inserting next CD, cancel and copy the Data files in their specified file directory. After that, 'register' the location with SpectraMove.

=====SCROLL FOR BETTER INSTRUCTIONS=====

To install atmosphere or trilogy open the setup (Trilogy_Setup.exe or Atmosphere_Setup.exe)
Then press Yes then click next then click on Yes then leave as is unless you know what you are doing then press Next then click Install. Now once the Data file Joiner Opens press Next and choose where to save the 3 gigabyte file this file will load all the sounds in the VST plugin once you have selected a directory for the 3 gigabyte data file press OK Once you see Please, press eject and insert CD marked Trilogy or Atmosphere press Cancel then press Finish.

Now get the data file for atmosphere or trilogy and paste it where you selected a directory for the 3 gigabyte file then use SpectraMove.exe as a Administrator then where is says Please select the product for which you would like to repair the registry Choose Trilogy: Repair or Atmosphere: Repair depending which VST plugin you are downloading then press the Select Data file and choose the Data file .dat you pasted then press Finish if an error says error while updating registry then you probably did not launch spectraMove as and Administrator or if it says key error then you probably did not install the VST correctly but if everything is good then you can launch your DAW and load up the VST
